# Coro19

## Build app

#### base build
```bash
$ make init-build
```

#### Set databses and migrations

Init lumen build for the first time
```bash
$ make init-build
```

Creation of database
```bash
$ make init-database
```

## Run app

#### Localy
```bash
$ make dev
```

## Serve app
```bash
$ make serve
```
or
```bash
$ php -S localhost:8000 -t public
```
